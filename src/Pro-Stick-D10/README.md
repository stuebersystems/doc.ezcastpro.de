# WLAN-Spezifikation

|  | EZCast Pro Stick II |
| :---- | :---- |
| WLAN-Standard | 802.11 ac 2T2R | 
| WLAN-Frequenz | 5 GHz: 5.150Ghz~5.825Ghz |
| Funkreichweite | Bis zu 10m |
| WLAN-Übertragungsrate |  866 MBit/s |
