# WLAN-Spezifikation

|  | EZCast Pro Box II |
| :---- | :---- |
| WLAN-Standard | 802.11ac 2T2R | 
| WLAN-Frequenzblock (5 GHz-Band) |  5150-5825 MHz |
| Funkreichweite | Bis zu 30m |
| WLAN-Übertragungsrate |  866 MBit/s |
